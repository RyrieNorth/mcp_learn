## 创建venv：
```bash
uv venv
```

## 应用虚拟环境：
```bash
source .venv/bin/activate
```

## 同步环境：
```bash
cd path/to/get_weather
uv sync
```

## 启动MCP Server：
```bash
uv run server.py
```
