## 创建venv：
```bash
uv venv
```

## 应用虚拟环境：
```bash
source .venv/bin/activate
```

## 同步环境：
```bash
cd path/to/fetch_time
uv sync
```

## 运行：
```bash
uv run path/to/fetch_time/main.py
```
